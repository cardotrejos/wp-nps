# Project Overview - FlowPulse

## What is FlowPulse?

FlowPulse is a **WhatsApp-native customer feedback platform** that enables businesses to collect NPS, CSAT, and CES surveys directly through WhatsApp conversations. Built to integrate with **Kapso WhatsApp Flows**, it provides a seamless, high-response-rate feedback collection experience.

## Target Market

- **Audience**: Small-to-Medium Businesses (SMBs)
- **Pricing**: SaaS model ($49-149/mo)
- **Use Case**: Customer satisfaction tracking, feedback collection, sentiment analysis

## Technical Summary

| Aspect | Details |
|--------|---------|
| **Type** | Turbo Monorepo |
| **Parts** | 2 apps + 4 packages |
| **Language** | TypeScript (strict) |
| **Stack** | Better-T Stack |
| **Frontend** | React 19 + Vite + TailwindCSS |
| **Backend** | Bun + Elysia + oRPC |
| **Database** | PostgreSQL + Drizzle ORM |
| **Auth** | Better Auth (email/password) |

## Getting Started

```bash
# Install dependencies
pnpm install

# Setup database
cp .env.example .env
# Edit .env with DATABASE_URL
pnpm db:push

# Start development
pnpm dev
```
